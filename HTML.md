# Que es HTML
*Siglas*
* Hyper Text *(es un vinculo, un enlace)*
* Markup *(marcado)*
* Language *(es un lenguaje)*

### HTTP & URL
* Hyper Transfer Protocol *(es el protocolo para transferir hyper texto, el protocolo a su vez es un conjunto de reglas que debes seguir, basicamente es el conjunto de reglas que debes seguir para mandarnos documentos web a traves de internet )*
* Uniform Resource Location *(es una direccion unica para encontrar un documento en todo internet)*

### W3C
* World Wide Web Consortium *(son las siglas de red de alcance global en inglés, se creo con el objetivo de fijar estandares que todos cumplan)*

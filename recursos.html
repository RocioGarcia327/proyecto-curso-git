Tipos de imagenes

Vectoriales

Son imagenes que no tienen resolucion, se pueden escalar infinitamente sin perder resolucion
Formatos: .svg

Mapa de bits

Pierden nitidez al acercarse, son puntitos llamados pixeles
Formatos: .jpg .jpeg (para fotos) .png (para fotos o imagenes detalladas con transparencia) .gif (para animaciones) .webp (formato liviano, soporta transparencia )

Unsplash es una pagina para encontrar imagenes gratis unDraw se puede personalizar un poco mas las imagenes

Pixeles de viewport (PV)

pixeles disponibles para mostrar contenido

Pixeles reales(PR)

Pixeles fisicos implementados por el fabricacnte en la pantalla

DevicePixelRatio

= PR / PV
